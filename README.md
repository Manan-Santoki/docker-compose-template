# docker-compose-sample
Docker Compose Sample

### Contains docker images:
- `RabbitMQ`
- `MongoDB`
- `Mysql`
- `Redis`
- `Solr`
- `Elasticsearch`
- `Kafka`

### Start
`cd /path/of/application`  
`docker-compose up`

### example
`cd redis`  
`docker-compose up`
